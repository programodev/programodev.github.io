```python
num1=int(input("Enter number 1: "))
num2=int(input("Enter number 2: "))
num1,num2=num2,num1
print(f"After swapping num1={num1} and num2={num2}")
```
<br>Output:
```python
Enter number 1: 20
Enter number 2: 10
After swapping num1=10 and num2=20
```
